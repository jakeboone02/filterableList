jQuery Filterable List Plugin

https://github.com/jakeboone02/filterableList

Initialization

CSS 
.hidden { display: none; }
.red { color: red; font-weight: bold; }
JavaScript 
$(function() {
    $( "ul.filterable" ).filterableList();
    $( "ul.paintable" ).filterableList({ hiddenClass: "red", placeholder: "Paint the town red..." });
});

Basic

ExampleMarkup
Search for "Test" or "this", etc.
  • Test
  • this
  • bad
  • boy
  • out
 
<ul class="filterable">
    <li>Test</li>
    <li>this</li>
    <li>bad</li>
    <li>boy</li>
    <li>out</li>
</ul>

Data filter

ExampleMarkup
Add data-filtertext to your list items to filter by that instead of the inner text. In this example, search for "you" or "won't", etc.
  • Test
  • this
  • bad
  • boy
  • out
 
<ul class="filterable">
    <li data-filtertext="You">Test</li>
    <li data-filtertext="won't">this</li>
    <li data-filtertext="believe">bad</li>
    <li data-filtertext="your">boy</li>
    <li data-filtertext="eyes">out</li>
</ul>
